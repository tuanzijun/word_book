## 生词本实现

### 功能
*****
* 添加生词
* 修改已有单词
* 删除单词
* （横屏）背单词状态（左列单词列表，点击右侧显示单词意思）
* contentProvider内容提供器
* API 调用（百度） 进行单词查询、内容翻译
